## Strategic Sequencing of Requests

### Phase 1 (Week 1-2): Broad Scope Requests
- Submit to DTRA, AFRL, DARPA simultaneously
- Target: Inter-agency collaboration records, program summaries
- Expected: ~40% release, 60% classified/exempt

### Phase 2 (Week 3-4): Specific Personnel Requests
- Submit to AFRL, MIT, AFIT
- Target: McCasland assignment history, program roles
- Expected: ~50% release (much unclassified), 50% privacy exemptions

### Phase 3 (Month 2): Declassification Review Requests
- Submit only if Phase 1-2 reveal significant classified material
- Target: Systematic declassification of time-sensitive documents
- Expected: 2-5 year timeline for sensitive materials

## Expected Classification Patterns

**Likely to be Classified:**
- Specific funding amounts for classified programs
- Operational/technical details of active military research
- Personnel security clearance levels or adjudication rationale
- Capabilities assessments or threat analysis
- Active program status or timelines

**Likely to be Released:**
- Personnel assignment titles and dates
- Inter-agency collaboration frameworks (unclassified portions)
- Published program summaries
- Organizational structure and reporting relationships
- Unclassified research summaries (2+ years old)

**Likely to be Redacted:**
- Personal privacy information (birthdates, SSNs)
- Deliberative process materials (internal debate)
- Attorney-client privileged communications
- Personal contact information

---

## Appeal Strategy

If requests are denied or heavily redacted:

1. **Request Detailed Vaughn Index** — Itemized exemption log
2. **Appeal within 30 days** — Cite exemption overreach, public interest
3. **Request Mandatory Review** — Argue declassification review is warranted
4. **Contact Ombudsman** — Agency FOIA ombudsman for informal resolution
5. **Consider Litigation** — If appeals unsuccessful (DOJ litigation section tracks FOIA suits)

---

## Practical Tips

✓ Submit requests in writing (email acceptable)  
✓ Use straightforward language (avoid jargon, keep requests focused)  
✓ Set response deadline clearly (20-30 days is standard)  
✓ Request fee waivers for public interest research  
✓ Save confirmation emails (FOIA tracking numbers)  
✓ Follow up if no response within 45 days  
✓ Request status updates monthly
