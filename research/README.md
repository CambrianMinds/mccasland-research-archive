# Research Workspace

This directory is the working analysis core of the repository.

## Primary files

- [`findings.md`](findings.md): current main brief
- [`trail-analysis.md`](trail-analysis.md): long-form synthesis of the technical, nuclear, and attribution trails
- [`timeline.md`](timeline.md): chronology
- [`findings-report.tex`](findings-report.tex): LaTeX source for the compiled report
- [`findings-report.pdf`](findings-report.pdf): compiled report output
- [`datasets/README.md`](datasets/README.md): generated analysis package and follow-up guides

## How to use this area

- Use `research/` for human-written synthesis and report drafting.
- Use `notes/` for claim-level or scratch analysis.
- Use `datasets/` for machine-generated analysis outputs, structured reports, and execution guides derived from the archive.
