# Artifacts

This directory holds generated or working artifacts that support the research repo but are not part of the public site itself.

## Layout

- [`evidence/`](evidence): loose evidence artifacts that used to live at the repo root
  - `xposts.md`
  - `TMBSPACESHIPS_Posts.xlsx`
  - `schematic.png`
- [`reports/`](reports): root-level report copies and exported report artifacts
- [`x-scrape/`](x-scrape): timestamped scraper outputs from `scripts/scrape_x_profile.py`

## Rule of thumb

- Put raw captured source pages in `sources/`.
- Put generated analysis in `research/datasets/`.
- Put ad hoc exports, working evidence blobs, and scraper runs here.
